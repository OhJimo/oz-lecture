⚙️ 주요 속성 테이블

속성명 적용 태그 설명 예시
action <form> 폼 데이터를 처리할 서버 URL action="/submit"
method <form> 데이터 전송 방식 (GET/POST) method="POST"
type <input> 입력 필드 유형 type="email"
name <input>, <select> 서버로 전송될 데이터 식별자 name="username"
id 모든 태그 요소의 고유 식별자 id="email-input"
value <input>, <option> 기본값 또는 전송값 value="kr"
for <label> 연결할 input의 id for="username"
placeholder <input> 입력 힌트 텍스트 placeholder="이름을 입력하세요"
min/max <input> 숫자/범위 최소/최대값 min="18" max="100"
controls <video>, <audio> 재생 컨트롤 UI 표시 controls
src <video>, <audio>, <source> 미디어 파일 경로 src="video.mp4"
width/height <canvas>, <video> 요소 크기 width="300" height="200"

🎯 HTML5 Input Types

Type 용도 특징
text 일반 텍스트 기본 한줄 입력
password 비밀번호 입력 내용 숨김
email 이메일 주소 이메일 형식 검증
date 날짜 선택 달력 UI 제공
number 숫자 입력 min/max 설정 가능
range 범위 슬라이더 시각적 범위 선택
checkbox 다중 선택 여러 항목 선택 가능
radio 단일 선택 그룹 중 하나만 선택
submit 폼 제출 폼 데이터 서버 전송
reset 폼 초기화 입력값 모두 리셋

🌐 HTTP 메서드

메서드 데이터 위치 특징 용도
GET URL에 포함 데이터 노출, 길이 제한 간단한 요청, 검색
POST 요청 본문 데이터 보안, 길이 제한 없음 회원가입, 로그인
